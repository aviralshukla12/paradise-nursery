# Paradise Nursery Shopping Application

## Project Name
**Paradise Nursery Shopping Application**

## Description
A dynamic React-based shopping cart application for Paradise Nursery, an online plant shop. The application allows users to browse houseplants, view details such as images, names, descriptions, and prices, add plants to a shopping cart, and manage cart items.

## Features
- Navigation bar with Home, Plants, Cart, and About Us links
- Product listing with categories and product cards
- Shopping cart with add/remove/increment functionality
- Redux state management for cart
- Responsive design with CSS styling

## Technologies Used
- React
- Redux Toolkit
- CSS
- JavaScript (ES6+)
